Launch the mnemonic app on the iOS simulator.

## Command

```bash
npm run ios
```

This runs `expo run:ios`, which compiles the native binary and starts Metro. The `ios/` native directory already exists (generated by a prior `expo prebuild`); no prebuild step is needed unless native dependencies change.

## Notes

- First build after a clean checkout takes ~2 minutes (Xcode compile + pod install).
- Subsequent launches are fast (Metro hot-reload only).
- If native dependencies change (new package with native code added), run `npx expo run:ios` directly — it will re-run pod install automatically.
- Do NOT use `node ios` or `npx node ios` — the `ios/` directory is a native Xcode project, not a Node module.
